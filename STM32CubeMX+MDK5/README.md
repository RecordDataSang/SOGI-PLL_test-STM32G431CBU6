## Usage

This project is provided in a minimal form. The full project files (Drivers, build files, etc.) are not included.

Please follow the steps below to regenerate and use the project.

### Requirements
- STM32CubeMX
- Keil MDK (MDK-ARM)

### Steps

1. Open the `.ioc` file using STM32CubeMX
2. Check the configuration (clock, GPIO, peripherals)
3. Click **"Generate Code"**
   - Toolchain: MDK-ARM (Keil)
4. After code generation, open the generated `.uvprojx` file in Keil
5. Build the project
6. Download and run on your target board

### Notes
- The `Drivers` and project files will be automatically generated by STM32CubeMX
- Make sure your STM32CubeMX and firmware package versions are compatible
- If any errors occur, try regenerating the project or updating the firmware package

### Optional
You may need to reconfigure:
- Toolchain path in Keil
- Debugger (ST-Link / J-Link)

## 使用方法

本项目仅提供最小工程（Core + .ioc），未包含 Drivers 和完整工程文件。

请按照以下步骤生成并使用工程。

### 环境要求
- STM32CubeMX
- Keil MDK（MDK-ARM）

### 使用步骤

1. 使用 STM32CubeMX 打开 `.ioc` 文件
2. 检查工程配置（时钟、GPIO、外设等）
3. 点击 **“生成代码（Generate Code）”**
   - 工具链选择：MDK-ARM（Keil）
4. 生成完成后，打开生成的 `.uvprojx` 工程文件
5. 编译工程
6. 下载程序到开发板运行

### 注意事项
- `Drivers` 及工程文件将由 STM32CubeMX 自动生成
- 请确保 CubeMX 版本及固件库版本匹配
- 若出现问题，可尝试重新生成代码或更新固件包

### 可选配置
首次使用可能需要手动配置：
- Keil 工具链路径
- 调试器（ST-Link / J-Link）
